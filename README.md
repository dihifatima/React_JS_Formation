# Les Bases de React

Bienvenue dans le projet **Les Bases de React** !  
Ce projet a pour objectif de découvrir et pratiquer les concepts fondamentaux de React.js, y compris les composants, les props, les hooks, et la gestion de l'état.

## 📂 Structure du projet

Le projet contient plusieurs sous-projets et exercices :

- `LesBaseDeReact` : Structure de base et premiers composants React.
- `LesCaptures` : Captures d’écran et illustrations des différents composants.
- `project_Hooks` : Exercices pratiques sur les hooks React (`useState`, `useEffect`, etc.).
- `project_Tp2` : Travaux pratiques sur les composants et la communication entre eux.
- `project_Tp4` : Exercices avancés sur les props, l'état et les événements.
- `projetTp5` : Projet final intégrant tous les concepts appris.

## 🛠 Technologies utilisées

- **React.js** : Bibliothèque JavaScript pour construire des interfaces utilisateur.
- **JavaScript (ES6+)** : Pour la logique côté client.
- **HTML & CSS** : Pour le rendu et le style des composants.
